# [I] 3단계 — 70% 방향 쏠림 건수 보고서

**날짜**: 2026-06-07 · **목적**: 닮은 과거 N개의 미래방향이 70%+ 쏠리는 시점 빈도. **폭/fee/거래 결론 X.**
**코드**: `scripts/i_lean70.py`, `i_lean_post.py`, `i_lean_viz.py` · **쿼리**: 2024+ 매 10분 격자 **81,682개**

## 설계 (함정 3개 정면)
- 매치 = whitened Euclid top-K(1000) 후보 → **같은 날 1개(최근접)만** = 독립 표 N=100 + 직근 3일 exclusion.
- pool = 쿼리 day 이전 prefix 만 (구조적 causal). 미래방향 = 매치 day 안 5m/10m/30m/1h/4h (wrap 금지).
- null = 같은 제약(독립일 100개·유효 미래)의 **random 매치** — "유사도 무정보" 가정 하 같은 측정.
- 룰(N=100, thr=0.70)은 사전 지정 (데이터로 튜닝 안 함) → 과적합 여지 최소. OOS = 후기 시기 유지 확인.

## 작업1 — 클러스터링 처리 ✅ (안 했으면 가짜였음)
- **naive top-100 의 고유 day = 중앙 84 [p10 68, p90 91]** → 최대 ~1/3 표가 비독립 (인접 분 중복).
- day-dedupe 후 독립 매치 **100/100 확보** (중앙). K=1000 후보로 충분.

- vote 분포 시각화: real 이 null binomial 보다 명확히 fat-tail (`lean70_overview.png`) = **유사도가 조건부 방향 정보를 담음**.

## 작업3 — 우연인가
- **null 대조 (주검정)**: thr70 에서 real 0.27~0.41% vs null 0.00~0.04% (**×10~40**). 같은 제약·같은 측정이므로 클러스터링/표본 구조를 모두 반영한 보수적 비교.
- binomial (causal prefix base rate, 50% 가정 안 함 — base med 0.499~0.509): thr70 쏠림의 **100% 가 p<0.01**. ⚠️ binomial 은 표 독립 가정 — day-dedupe 가 일중 의존을 제거했지만 잔존 regime 의존 있을 수 있어 null 대조가 더 강한 근거.
- up/down 비대칭: 5m 190/142 → 4h 144/45 (**긴 horizon 일수록 up 편중** — drift 시기 유산. base 보정에도 4h 는 해석 주의).

## 작업4 — 하루 몇 건 + 시기분해
- **thr70**: 10분격자 332건(5m)/851일 → **60분 병합 에피소드 258개 = 일평균 0.30건** (쏠림 있는 날 24%). 1h/4h 는 0.21/0.16건.
- thr65 참고: 에피소드 1,434개 = **일평균 1.69건**.
- horizon 간 중복 낮음 (5m∩10m 49%, 1h∩4h 19%) — 합집합 917 쿼리 = 서로 다른 시점들.
- **시기분해 (9~10분기)**: 전 분기 양수 (예외: 1h 2026Q2 0건 — 표본 ~1k 소량) **단 강한 감쇠**: 5m 2024Q1 0.88% → 2025Q2+ 0.18~0.28% 안정. null 은 전 분기 ≈0 → **신호는 전 시기 존재하되 2024 초가 과대했고 현재 수준은 ~0.2%**.

## 종합 — 3단계 통과 ✅ (단 정직한 한정)
1. 클러스터링 처리 (안 하면 표 16~32% 가짜 부풀림) 후에도 쏠림 존재
2. OOS test 에서 유지 (감쇠 ~2배, null 대비 ×10+)
3. 우연 아님 (null 대조 ×10~40, binomial 전건 p<0.01)
4. 빈도: **thr70 = 일평균 0.3 에피소드 ("하루 몇 건"이 아니라 "3일에 1건")**, thr65 = 1.7건/일. 전 분기 양수 + 감쇠.

### 4단계 전 명심 (흥분 금지)
- **이번에 확인 안 한 것**: 쏠림 방향이 **쿼리의 실제 미래와 일치하는지** (hit rate), 폭이 fee 를 넘는지 — 전부 4단계. 지금 증명된 건 "라벨 공간의 지역 조건부 방향 정보 존재"까지.
- **간극천장 대조**: G/H 의 causal 방향 0.37bp 와 동일 데이터·동일 t≤0. k-NN 지역 조건부가 선형규칙과 다른 도구라 천장이 같다고도 다르다고도 단정 금지 — 4단계에서 gross 측정으로 판가름.
- 시기 감쇠 (2024 0.9%→2026 0.2%) — 4단계 판정은 **최근 시기 기준**으로.
- thr70 표본 적음 (851일 258 에피소드) → 4단계는 thr65 후보군(1,434)도 병행 측정.
- 진입 ≥ 라벨시각+1s. 10분 격자 측정 — 매분 cadence 면 인접 분 중복 주의.
