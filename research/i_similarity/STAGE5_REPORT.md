## 작업1 — walk-forward 구조 (OOS 표본 최대화)
- 인식: pool = 쿼리 과거 prefix + 룰(N100/thr70) 사전지정 + whitening 2023 fit → **모든 2024+ 쿼리(81,682)가 구조적 OOS**. 이전 "train/test" 는 보고용 분할이었음.
- 순수성 점검 2개: ①대표 21차원 선택(전체기간 corr 사용) → 21차원 상관구조 2023 vs 전체 **max|Δr| 0.194, med 0.012, >0.2 쌍 0/210** + 203일↔전체 동일 대표 재현 — 오염 무시 가능 ②"30m/1h/4h 생존" 은 사후 선택 → **Bonferroni 99.5% (10셀)** 로 보정.

## 작업4 — thr 단조 (0.60~0.78 촘촘): **매끄러움 = 구조** ✓
- hit: 0.54→0.56→0.58→0.61→0.64→0.66→… (30m 열) 톱니 없음. gross 연속 상승 (+0.9→+155.7). thr65 죽음/70 생존의 정체 = **gross 가 fee 11bp 를 thr≈0.67 에서 연속 통과** — 절벽/우연 아님.

## 작업5 — horizon 촘촘 (15m~6h): **매끄럽지 않음 — 거친 증가 + 2h/3h 골 + 4h 돌출** (정직)
- net day-mean: 5m -3.1 / 10m +1.3 / 15m -4.7 / 20m +0.9 / 30m +8.7 / 45m +21.0 / 1h +27.7 / **2h +16.9 / 3h +1.6 (골, hit-edge 0.07~0.09)** / 4h +74.7 / 6h +25.5 (n=27).
- 사전등록 가설 채점: "짧은 horizon 노이즈/fee 잠식" **확증**, "~4h 컨텍스트 한계 후 감쇠" 는 6h n=27 로 **미검증**, 2h/3h 골은 **예상 못 한 구조** (열린 질문 — 시간대 효과/세션 구조 후보).
- **4h 돌출의 confounder 점검 (시간대)**: 4h 는 mod≤1199(이른 시간) 쿼리만 가능. 30m/1h 를 같은 시간대로 제한하면 **오히려 약해짐** (30m: 이른 +8.1 vs 늦은 +39.1; 1h: +37.0 vs +69.8) → 4h 는 불리한 시간대에서 +101 달성 = **돌출이 시간대 인공물 아님** (오히려 보수적).
- 부수 발견: 늦은 시간대(US 세션) lean 이 더 강함 (30m hit 0.71/+39, 1h 0.71/+70) — 사후 subgroup 이므로 결론 아님, 관찰만 기록.

## 작업6 — why (분해: `why_horizon_decomp.csv`, `why_horizon.png`)
1. **fee 고정비 vs |move| 스케일 (구조적·확실)**: median |move| 17bp(5m)→115bp(4h), **fee 11bp = 5m 이동의 65% → 4h 의 10%**. 짧은 horizon 은 hit 가 완벽해도 fee 가 잠식 — 구조적 불가.
2. **hit-edge 도 horizon 과 함께 상승 (예측 성분)**: edge 0.06(5m)→0.27(1h)→0.37(4h) (단 2h/3h 골). **net↑ = 폭(×7) × 예측(×6) 둘 다** — 폭만으로 설명 안 됨.
3. lean 순간 = 고변동 순간: lean 이벤트 |move| ≈ 시장 중앙의 ~2배 (17 vs 9.5bp 등) — 쏠림은 "큰 움직임이 진행 중인 상태" 에서 발생.
